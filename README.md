## סטאק טכנולוגי
- **HTML יחיד** (`index.html`) — כל ה-CSS וה-JS inline.
- **Tailwind CSS** דרך CDN (עם קונפיג מותאם — צבעי מותג, גופנים, צללים).
- **גופנים:** Rubik (כותרות) + Heebo (גוף), מ-Google Fonts.
- **Canvas 2D** לאפקט פס הריתוך; **`<video>`** לרקע ה-Hero.
- **ללא בק-אנד** — הטופס נשלח דרך Formspree, פניות דרך WhatsApp/טלפון.
- כלי פיתוח: שרת סטטי (`serve.mjs`) + צילומי מסך (`screenshot.mjs` עם Puppeteer).

**צבעי מותג:** בסיס `#0E0F12` · surface `#16181D` · אקסנט כתום-נחושת `#FF6A21` ·
אקסנט קשת כחול `#2BD9FF`.

## פיצ'רים מיוחדים

### 🔥 אפקט "פס הריתוך" (Canvas 2D)
קנבס קבוע (`#weld-fx`) בצד שמאל מצייר תפר ריתוך מתפתל שמתקדם **לפי הגלילה**: ראש קשת
לוהט יורד במורד הדף, התפר מאחוריו זוהר (stacked-dimes), גיצים מתפרצים, ובסוף הדף
הקשת **נכבית**. עובד בשני כיווני הגלילה. הקוד נמצא בתחתית `index.html` (חיפוש: `WELD ENGINE`).

### 🎬 וידאו Hero רספונסיבי
שני וידאו (דסקטופ 16:9 / מובייל 3:4 פורטרט) — **רק המתאים נטען**, ומתחלף בזמן אמת
כשחוצים את ה-breakpoint (768px). `autoplay muted loop playsinline`, עם תמונת poster
כגיבוי. החלפת המקור: בלוק ה-JS `heroVideo` בתוך הסקריפט הראשי.

### 🖼️ גלריה (masonry) + הוספת תמונות
הגלריה נבנית אוטומטית ממערך **`GALLERY`** בתוך `index.html` (חיפוש: `רשימת הגלריה`).
להוספה: שמים קובץ בתיקיית `gallery/` ומוסיפים שורה:
```js
{ src: 'gallery/שם-הקובץ.jpg', title: 'תיאור קצר' },
```
הוראות מלאות: [`gallery/קרא-אותי-הוראות.txt`](gallery/קרא-אותי-הוראות.txt).

### ♿ ווידג'ט נגישות (תקן IS 5568 / WCAG 2.0 AA)
`accessibility-widget.js` מזריק כפתור נגישות (כחול, פינה שמאלית-תחתונה) ופאנל עם:
גודל טקסט, ניגודיות, עצירת אנימציות (כולל הווידאו ופס הריתוך), גופן דיסלקציה,
הדגשת קישורים/כותרות, מדריך קריאה, סמן גדול, ועוד. כולל **עמוד הצהרת נגישות**
(`accessibility.html`) עם פרטי רכז הנגישות.

## הרצה מקומית ופיתוח

### התקנה
```bash
npm install            # אם נכשל על הרשאות cache:
npm install --cache ./.npm-cache
```

### הרצת השרת
```bash
node serve.mjs         # מגיש את האתר ב-http://localhost:3000
```
> `serve.mjs` תומך ב-**HTTP Range requests** (תגובת 206) — חובה כדי שהווידאו ינוגן
> ב-Safari ובדפדפנים מחמירים.

### צילומי מסך
```bash
node screenshot.mjs http://localhost:3000 [label] [--mobile]
```
נשמרים ל-`temporary screenshots/`.
> **הערה סביבתית:** Chrome המצורף ל-Puppeteer לא תקין במכונה הזו, לכן `screenshot.mjs`
> מצביע ל-**Chrome המערכת** (`/Applications/Google Chrome.app/...`). לפריסות עם `100svh`
> + WebGL, הצילום מקבע את גובה ה-Hero כדי למנוע כפילות בלכידה.
